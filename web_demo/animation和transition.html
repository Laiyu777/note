使用不同方法的动画效果

使用hover

								
.box{
	height: 200px;
	width: 200px;
	background: red;
	transition: all 2s;
}
.box:hover{
	width: 500px;
	background: yellow;
}

使用animation

animation的优先级是大于transtion的，就是当设置animation的时候，transition会失效 


.box2{
	width: 200px;
	height: 200px;
	background-color: red;
	transition: all 3s;/*这行没有作用*/
}
.box2:hover{
	-webkit-animation:myfirst 3s;
	background: black;/*这行没有作用*/
}
@-webkit-keyframes myfirst{
	from{
		background: green;width: 100px/*覆盖了原来的样式*/
	}
	to{
		background: pink;width: 300px;/*覆盖了原来的样式*/
	}
}

使用jquery的hide(mills)和show(mills)


$("#hide").on("click",function(){
	$(this).prev().hide(2000)	
})
$("#show").on("click",function(){
	$(this).prev().prev().show(2000)
})

使用display+animation产生动画效果(需要jquery)

在keyframes中设置transform:scale(0)-->(1)，当display变为block的时候，就会执行这个动画

transform:scale这个值在元素中默认是1,其他值是none,意思是回到css中写的样式

display的高度还是马上显示出来的,可以在from和to中设置heght:0px--->100px;


.box4{
	width: 200px;
	height: 200px;
	background-color: red;
	animation: second 2s;
	display: none;			
	background: rgba(0,0,0,0.2);
}
@-webkit-keyframes second{
	from{transform: scale(0);}
	to{transform: scale(1);}
}
			

$("#modal_btn").click(function(){
	$(this).prev().css("display","block");
})
